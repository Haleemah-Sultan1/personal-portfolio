# Static assets — drop your images here

Copy your screenshots and book cover into this folder with these exact names:

| File | Used for |
|------|----------|
| `book-cover.png` | The Sky We Own book section |
| `tumble-pop.png` | Tumble Pop game project |
| `metal-slug.png` | Metal Slug-style game project |
| `aakhri-mor.png` | Aakhri Mor game project |

Supported formats: `.png`, `.jpg`, `.webp` — update the path in `content.json` if you use a different extension.
